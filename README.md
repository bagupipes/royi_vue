## 术语展示
<类型>[范围]: <描述>

| 类型         | 说明                   | 示例                      |
| ---------- | -------------------- | ----------------------- |
| `feat`     | **新功能**（Feature）     | `feat: 新增博物馆预约功能`       |
| `fix`      | **修复bug**​           | `fix: 解决展品图片上传失败问题`     |
| `docs`     | **仅文档更新**​           | `docs: 更新项目README安装说明`  |
| `style`    | **代码风格调整**（不影响逻辑）    | `style: 调整登录页按钮颜色`      |
| `refactor` | **重构**（非新增功能也非修复bug） | `refactor: 优化展品查询SQL性能` |
| `test`     | **增加或修改测试用例**​       | `test: 为预约模块添加单元测试`     |
| `chore`    | **杂项**（构建工具、依赖更新等）   | `chore: 升级webpack至最新版本` |
